## Introduction

The FileCard component displays uploaded or selected files in an attractive card format with thumbnails, metadata, and action buttons.

## Installation

```bash
pnpm dlx shadcn@latest add @shelf-ui/file-card
```

## Usage

```tsx
import { FileCard } from "@/components/ui/file-card"

export function Example() {
  return (
    <FileCard
      file={{
        name: "document.pdf",
        size: 1024000,
        type: "application/pdf"
      }}
      onRemove={() => console.log('removed')}
    />
  )
}
```
