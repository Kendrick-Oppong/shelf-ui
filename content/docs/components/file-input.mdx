## Introduction

The FileInput component provides a styled alternative to the native file input, maintaining full accessibility while offering better visual customization.

## Installation

```bash
pnpm dlx shadcn@latest add @shelf-ui/file-input
```

## Usage

```tsx
import { FileInput } from "@/components/ui/file-input"

export function Example() {
  return (
    <FileInput
      onChange={(files) => console.log(files)}
      placeholder="Choose files..."
    />
  )
}
```
