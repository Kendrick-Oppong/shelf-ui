# Shelf UI

A shadcn-style file UI component library. Copy-paste components for the complete file experience — upload, preview, manage, navigate — with first-class integrations for Supabase Storage, AWS S3, and Cloudinary.

## Documentation

[shelfui.dev](https://shelfui.dev) *(coming soon)*

## Installation

```bash
npx shadcn@latest add @shelf-ui/dropzone
```
